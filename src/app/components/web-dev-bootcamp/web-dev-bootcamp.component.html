25: EJS (Embedded JavaScript templating)
  • Keywords

    Install with: npm i ejs

  • Tags

    <% 'Scriptlet' tag, for control-flow, no output

    <%_ ‘Whitespace Slurping’ Scriptlet tag, strips all whitespace before it. (JS Execute)

    <%= Outputs the value into the template HTML escaped. (JS Output)

    <%- Outputs the unescaped value into the template. (Render HTML)

    <%# Comment tag, no execution, no output. (Stop Execution)

    <%- include() Insert another EJS file

    <%% Outputs a literal '<%'

    %> Plain ending tag

    -%> Trim-mode ('newline slurp') tag, trims following newline

    _%> ‘Whitespace Slurping’ ending tag, removes all whitespace after it

  • Templates, Partials and Layouts

    To charge statics files like images, js files etc app.use(express.static("public"));

    To charge statics files like images, js files etc <%- include("partials/header.ejs") %>
